# 第9课学习记录：电的基础

## 日期
2026-07-10

## 核心知识点

1. **电荷与静电**：
   - 两种电荷：正电荷（丝绸摩擦玻璃棒）、负电荷（毛皮摩擦橡胶棒）
   - 同种电荷互相排斥，异种电荷互相吸引
   - 元电荷 e = 1.6 × 10⁻¹⁹ C
   - 摩擦起电本质是电子转移

2. **电流**：
   - 电流是电荷的定向移动形成的
   - 规定：正电荷定向移动的方向为电流方向（金属中电子移动方向与电流方向相反）
   - 单位：A（安培），1A = 1000mA = 10⁶μA
   - 电流表：串联使用，绝对不能直接接电源两极

3. **电压**：
   - 电压是形成电流的原因，电源是提供电压的装置
   - 单位：V（伏特），1kV = 1000V
   - 常见电压：干电池1.5V、家庭电路220V、安全电压不高于36V
   - 电压表：并联使用，可以直接接电源两极

4. **电阻**：
   - 电阻表示导体对电流的阻碍作用，符号R，单位Ω
   - 影响因素：材料、长度（正比）、横截面积（反比）、温度
   - 导体的电阻是其本身性质，与是否通电无关
   - 变阻器：通过改变接入电路中电阻丝长度来改变电阻

5. **欧姆定律**：
   - 公式：I = U/R
   - 对同一导体（R不变）：I与U成正比
   - 对同一电压（U不变）：I与R成反比
   - 变形：U = IR，R = U/I

6. **串联电路**：
   - 电流处处相等：I = I₁ = I₂
   - 总电压等于各部分之和：U = U₁ + U₂
   - 总电阻等于各电阻之和：R = R₁ + R₂
   - 用电器互相影响

7. **并联电路**：
   - 各支路电压相等：U = U₁ = U₂
   - 总电流等于各支路之和：I = I₁ + I₂
   - 总电阻倒数等于各电阻倒数之和：1/R = 1/R₁ + 1/R₂
   - 用电器互不影响

8. **电功与电功率**：
   - 电功：W = UIt = Pt
   - 电功率：P = W/t = UI
   - 额定功率 vs 实际功率

9. **焦耳定律**：
   - Q = I²Rt
   - 利用电热：电热水器、电饭煲
   - 防止电热危害：散热风扇、散热孔

10. **安全用电**：
    - 不高于36V为安全电压
    - 家庭电路顺序：进户线→电能表→总开关→保险装置→用电器
    - 电流过大原因：短路、过载
    - 开关接在火线上
    - 保险丝不能用铜丝/铁丝代替

## 易错点

- ❌ 误以为"电流方向是电子移动方向"（相反！）
- ❌ 电流表直接接电源两极（会短路烧坏！）
- ❌ 认为"电阻随电压增大而增大"（电阻是导体本身性质）
- ❌ 并联总电阻比任一支路电阻大（并联总电阻更小！）
- ❌ 用铜丝代替保险丝（铜丝熔点高，不起保护作用）
- ❌ 开关接在零线上（应接在火线上才安全）
- ❌ 两电阻并联 R = R₁ + R₂（应为 R = R₁×R₂/(R₁+R₂)）
- ✅ 记忆口诀："串等流压相加，并等压流相加"、"一上一下接变阻"

## 考试高频考点

2. 电流表与电压表的使用区别
3. 常见电压值（干电池1.5V、家庭220V、安全≤36V）
4. 欧姆定律的计算与应用
5. 串联与并联电路的电流、电压、电阻特点
6. 家庭电路的连接顺序与安全用电
7. 滑动变阻器的接法和作用
