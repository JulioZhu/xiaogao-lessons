# 第10课学习记录：化学方程式入门

## 日期
2026-07-11

## 核心知识点

1. **质量守恒定律**：
   - 参加化学反应的各物质的质量总和 = 生成的各物质的质量总和
   - 微观本质：反应前后原子的种类、数目、质量都不变
   - 分子种类一定改变，分子数目可能改变
   - 元素种类一定不变

2. **化学方程式的含义**：
   - 用化学式表示化学反应的式子
   - 三层含义：质的方面（谁和谁反应生成什么）、粒子数方面（分子个数比）、质量方面（质量比）
   - 符号意义：+（和）、→（生成）、↑（气体）、↓（沉淀）、△（加热）

3. **化学方程式的配平**：
   - 依据：质量守恒定律（左右各元素原子数相等）
   - 方法：最小整数配平法（找→定→调→查）
   - 口诀：左写反应物，右写生成物；写好化学式，系数要配平

4. **四种基本反应类型**：
   - 化合反应：A + B → AB（多变一）
   - 分解反应：AB → A + B（一变多）
   - 置换反应：A + BC → AC + B（单质换单质）
   - 复分解反应：AB + CD → AD + CB（成分互换）

5. **常见气体的实验室制法**：
   - O₂：2KClO₃ --(△,MnO₂)--> 2KCl + 3O₂↑
   - CO₂：CaCO₃ + 2HCl --> CaCl₂ + H₂O + CO₂↑
   - H₂：Zn + H₂SO₄ --> ZnSO₄ + H₂↑

## 易错点

- ❌ 混淆"点燃"和"燃烧"（条件是点燃，现象是燃烧）
- ❌ ↑↓使用不当（反应物已有气体时不标↑）
- ❌ 把质量守恒理解为体积守恒
- ❌ 化学式写错导致整个方程式错误
- ❌ 忘记写反应条件
