光合作用与呼吸作用

这两个过程是地球上所有生命活动的能量基础。

学习目标

学完本课，你应该能够：

  1. 写出光合作用和有氧呼吸的反应式
  2. 描述光合作用的两个阶段及其场所
  3. 理解呼吸作用的三个阶段
  4. 对比分析光合作用与呼吸作用的异同
  5. 解释影响光合速率和呼吸速率的因素

一、光合作用——生命的能量之源

光合作用是绿色植物、藻类和某些细菌利用光能，将二氧化碳和水转化为有机物（主要是葡萄糖）并释放氧气的过程。它是地球上几乎所有生命能量的最终来源。

☀️ 光合作用总反应式
6CO₂ + 12H₂O ──[光能, 叶绿体]──> C₆H₁₂O₆ + 6O₂ + 6H₂O

光合作用在叶绿体中进行，分为两个阶段：

1. 光反应（光依赖反应）

场所：类囊体薄膜
H₂O → O₂↑ + [H] + ATP

需要光、色素、酶

光反应的要点：

  • 水的光解：2H₂O → 4[H] + O₂（氧气来源于水，不是 CO₂！考试高频考点）
  • ATP 的合成：ADP + Pi + 能量 → ATP（光能转化为 ATP 中活跃的化学能）
  • NADPH 的生成：[H] 作为还原剂，参与暗反应

2. 暗反应（碳反应 / Calvin 循环）

场所：叶绿体基质
CO₂ + C₅ → 2C₃
C₃ ──[ATP, [H]]──> C₆H₁₂O₆ + C₅再生

不需要光，但需要光反应的产物

暗反应包含两个关键步骤：

  1. CO₂ 的固定：CO₂ + C₅ → 2C₃（由 RuBisCO 酶催化，是地球上含量最丰富的蛋白质）
  2. C₃ 的还原：C₃ 接受 ATP 提供的能量和 [H]，被还原为糖类，同时再生 C₅
🔑 关键理解
"暗反应"不等于"不需要光"——暗反应本身不消耗光能，但它需要光反应产生的 ATP 和 [H]。所以在黑暗中，暗反应会因为缺乏 ATP 和 [H] 而很快停止。

更准确的说法是"碳反应"，因为它的核心是把 CO₂ 转化为有机物。

二、呼吸作用——能量的释放者

呼吸作用是细胞将有机物（主要是葡萄糖）氧化分解，释放能量并生成 ATP 的过程。所有活细胞时刻都在进行呼吸作用。

🫁 有氧呼吸总反应式
C₆H₁₂O₆ + 6O₂ ──[酶]──> 6CO₂ + 6H₂O + 大量能量（约 38ATP）

三个阶段

阶段 场所 主要反应 产物 能量
第一阶段
糖酵解		 细胞质基质 C₆H₁₂O₆ → 2丙酮酸 + [H] 丙酮酸、[H] 少量（2ATP）
第二阶段
三羧酸循环		 线粒体基质 丙酮酸 + H₂O → CO₂ + [H] CO₂、[H] 少量（2ATP）
第三阶段
电子传递链		 线粒体内膜 [H] + O₂ → H₂O H₂O 大量（34ATP）
💡 记忆口诀
「一糖二丙三水，能量逐步释放」
第一阶段在胞质，葡萄糖变丙酮酸；
第二阶段在线粒基质，丙酮酸变二氧化碳；
第三阶段在线粒内膜，氢氧结合生成水，能量最多在这里！

无氧呼吸

当氧气不足时，细胞可以进行无氧呼吸（发酵）：

🔬 无氧呼吸
C₆H₁₂O₆ ──[酶]──> 2C₂H₅OH（酒精）+ 2CO₂ + 少量能量（2ATP）
或
C₆H₁₂O₆ ──[酶]──> 2C₃H₆O₃（乳酸）+ 少量能量（2ATP）

常见类型：

  • 酒精发酵：酵母菌、植物根部缺氧时（如水稻田）
  • 乳酸发酵：乳酸菌、动物肌肉细胞剧烈运动时
⚠️ 考试高频陷阱
无氧呼吸只在第一阶段释放能量！第二阶段（丙酮酸转化为酒精或乳酸）不产生 ATP。所以无氧呼吸总共只产生 2ATP，远少于有氧呼吸的 38ATP。

三、光合作用 vs 呼吸作用——对比分析

比较项目 光合作用 呼吸作用
发生部位含叶绿体的细胞所有活细胞
发生条件需要光有光无光均可
物质变化合成有机物（CO₂→有机物）分解有机物（有机物→CO₂）
能量变化光能→化学能（储能）化学能→ATP+热能（放能）
气体交换吸收 CO₂，释放 O₂吸收 O₂，释放 CO₂
实质合成有机物，储存能量分解有机物，释放能量
🔗 两者的关系
光合作用和呼吸作用是相互依存、对立统一的关系：

对立：一个合成有机物，一个分解有机物；一个吸 CO₂ 放 O₂，一个相反。

依存：光合作用产生的有机物和 O₂ 是呼吸作用的原料；呼吸作用产生的 CO₂ 和 ATP 可为光合作用所用。

在白天，植物同时进行光合作用和呼吸作用。当光合速率 > 呼吸速率时，表现为吸收 CO₂、释放 O₂。

四、影响光合速率的因素

1. 光照强度

在一定范围内，光合速率随光照强度增加而加快。达到光饱和点后不再增加。

📊 三个关键点
呼吸点：光照为零，只进行呼吸作用，释放 CO₂
光补偿点：光合速率 = 呼吸速率，净光合为零
光饱和点：光合速率达到最大，再增加光照也不再提高

2. CO₂ 浓度

CO₂ 是光合作用的原料。在一定范围内，光合速率随 CO₂ 浓度增加而加快。达到CO₂ 饱和点后不再增加。

3. 温度

温度通过影响酶的活性来影响光合速率。大多数植物在 25~30°C 时光合速率最高。温度过高或过低都会降低酶活性。

4. 水分

水是光合作用的原料之一。缺水会导致气孔关闭，减少 CO₂ 进入，从而降低光合速率。

🌱 农业生产中的应用
增加光照强度、提高 CO₂ 浓度（施用有机肥、通风）、合理灌溉、适当升温（温室大棚）——这些都是为了提高光合效率，增加农作物产量。

五、易错点总结

❌ 常见错误 ✅ 正确理解
光合作用释放的 O₂ 来自 CO₂ O₂ 全部来自水的分解（同位素标记实验证明）
暗反应不需要光 暗反应不需要光能直接参与，但需要光反应产物（ATP 和 [H]），间接依赖光
植物白天只进行光合作用 白天同时进行光合和呼吸，只是光合速率通常更大
无氧呼吸第二阶段也产能 只有糖酵解阶段产 ATP，丙酮酸转化阶段不产 ATP
呼吸作用就是"喘气" 呼吸作用是细胞内的化学反应，"喘气"（肺通气）只是为呼吸作用提供 O₂
所有植物细胞都能光合作用 只有含叶绿体的细胞才能，如根细胞不能

六、与小学教学的联系

作为小学科学教师，你需要向学生解释这些概念。以下是教学建议：

🏫 教学建议
光合作用：可以用"植物是小厨师，阳光是炉火，CO₂和水是食材，做出食物（糖）并放出氧气"来比喻。让学生观察植物在光照和黑暗中的生长差异。

呼吸作用：强调"所有活着的生物都要呼吸"——不仅是动物，植物也要呼吸！可以做种子萌发实验，用温度计测量萌发种子释放的热量来证明呼吸作用产生能量。

📝 课堂练习
1. 光合作用释放的氧气来源于：
✅ 正确！O₂ 全部来自水的光解。1941 年鲁宾和卡门用同位素¹⁸O 标记实验证明了这一点。
❌ 不对。光合作用释放的 O₂ 来自水（H₂O）的光解，不是 CO₂。这是考试高频考点！
2. 有氧呼吸产生 ATP 最多的阶段是：
✅ 正确！第三阶段在线粒体内膜上进行，[H] 与 O₂ 结合生成水，释放大量能量，产生约 34 个 ATP。
❌ 不对。第三阶段（电子传递链）产生约 34 个 ATP，占总量的 90% 以上。第一、二阶段各只产生 2 个 ATP。
3. 下列关于"暗反应"的说法正确的是：
✅ 正确！暗反应（碳反应）在叶绿体基质中进行，虽然不直接需要光，但需要光反应产生的 ATP 和 [H] 作为能量和还原剂。
❌ 不对。暗反应在叶绿体基质中进行，需要光反应提供的 ATP 和 [H]，在白天和黑夜都可以进行（只要有 ATP 和 [H] 供应）。
4. 植物根部细胞在缺氧条件下主要进行：
✅ 正确！大多数植物细胞（包括根部）在缺氧时进行酒精发酵，产生酒精和 CO₂。这也是农田积水会导致根系腐烂的原因。
❌ 不对。植物细胞（包括根部）缺氧时主要进行酒精发酵，产生酒精和 CO₂。动物肌肉细胞缺氧时才进行乳酸发酵。
5. 在光补偿点时，植物的状态是：
✅ 正确！光补偿点是光合速率恰好等于呼吸速率时的光照强度。此时植物净光合为零，不积累有机物也不消耗有机物。
❌ 不对。光补偿点的定义是光合速率 = 呼吸速率。此时植物既在进行光合作用也在进行呼吸作用，两者速率相等。
