遗传与变异基础

为什么孩子像父母又不完全相同？遗传是生命的"复制密码"，变异是进化的"原材料"。

学习目标

学完本课，你应该能够：

  1. 说出DNA、基因和染色体的关系
  2. 理解人的染色体数目和性别决定
  3. 解释孟德尔遗传规律（分离定律）
  4. 区分可遗传变异和不可遗传变异
  5. 理解遗传病的基本概念和预防措施

一、DNA、基因与染色体

遗传信息的载体是一个多层次的结构体系，从大到小依次是：染色体 → DNA → 基因。

染色体

染色体存在于细胞核中，容易被碱性染料染成深色。它是遗传物质的载体。人的体细胞中有23对（46条）染色体。

🧬 染色体 = DNA + 蛋白质
染色体是由DNA和蛋白质两种物质组成的复合结构

DNA

DNA（脱氧核糖核酸）是主要的遗传物质，呈双螺旋结构。它像一个长长的梯子，两条长链盘旋而成。

碱基互补配对原则
A — T
│  │
T — A
A=腺嘌呤，T=胸腺嘧啶，G=鸟嘌呤，C=胞嘧啶
G — C
│  │
C — G

基因

基因是具有遗传效应的DNA片段。一个DNA分子上有多个基因。基因控制生物的性状。

📐 层次关系（必考）
细胞核 → 染色体 → DNA → 基因
数量关系：1个体细胞 → 23对染色体 → 46个DNA → 数万个基因
位置关系：基因在DNA上，DNA在染色体上，染色体在细胞核中
💡 形象比喻
如果把染色体比作一本书：
- 书 = 染色体
- 书页 = DNA
- 一个个段落/章节 = 基因
- 每个段落传递不同的信息（控制不同的性状）

二、人的染色体与性别决定

人的体细胞中有23对染色体，其中22对是常染色体，1对是性染色体。

常染色体与性染色体

类型数量说明
常染色体22对（44条）男女都一样，与性别无关
性染色体女性：XX两条形态相同，用XX表示
男性：XY两条形态不同，用XY表示

生殖细胞中的染色体

在形成精子或卵细胞的减数分裂过程中，染色体数目减半。

👨 男性产生精子
22对+XY → 22条+X + 22条+Y
产生两种精子，比例1:1
👩 女性产生卵细胞
22对+XX → 22条+X
只产生一种卵细胞
🎯 核心考点
生男生女的概率：由于两种精子与卵细胞结合的机会均等，因此生男生女的比例接近 1:1。
决定性别的是父亲的精子类型（X精子还是Y精子），与母亲无关！

三、基因的显性和隐性

控制同一性状的不同表现形式称为相对性状（如双眼皮vs单眼皮）。每个性状由一对基因控制。

等位基因

  • 显性基因：用大写字母表示（如 A），控制的性状为显性性状
  • 隐性基因：用小写字母表示（如 a），控制的性状为隐性性状
🧪 基因型与表现型
AA
纯合显性 → 显性性状
Aa
杂合 → 显性性状（携带隐性基因）
aa
纯合隐性 → 隐性性状
⚠️ 关键理解
Aa 中，显性基因A掩盖了隐性基因a的作用，所以表现为显性性状。但a并没有消失，仍然可以传递给后代。这就是为什么父母都是双眼皮（Aa），孩子可能是单眼皮（aa）。

孟德尔分离定律

在形成生殖细胞时，成对的基因彼此分离，分别进入不同的生殖细胞中。

📊 经典杂交示例：双眼皮(Aa) × 单眼皮(aa)
父方基因型 Aa（双眼皮）
母方基因型 aa（单眼皮）
后代基因型 Aa : aa = 1 : 1
后代表现型 双眼皮 : 单眼皮 = 1 : 1

四、遗传病与优生优育

人类遗传病

由遗传物质改变引起的疾病，称为遗传病。主要分为：

类型原因举例
单基因遗传病一对等位基因异常白化病、色盲、多指、血友病
多基因遗传病多对基因+环境因素高血压、糖尿病、唇裂
染色体异常遗传病染色体数目/结构异常21三体综合征（唐氏综合征）、猫叫综合征
🏥 优生优育措施
  1. 禁止近亲结婚：近亲携带相同隐性致病基因的概率大大增加
  2. 遗传咨询：了解家族病史，评估遗传风险
  3. 产前诊断：B超检查、羊水检查、基因检测等
  4. 适龄生育：女性最佳生育年龄24-29岁
🔬 近亲结婚的危害
直系血亲和三代以内的旁系血亲禁止结婚。因为近亲之间携带相同隐性致病基因的可能性大，婚后子女患遗传病的概率显著增加。

例如：表兄妹结婚，子女患隐性遗传病的概率是一般人群的数倍至数十倍。

五、变异的类型与来源

变异的分类

类型能否遗传原因举例
可遗传变异能遗传物质发生改变基因突变、基因重组、染色体变异
不可遗传变异不能环境因素影响，遗传物质未变晒黑的皮肤、锻炼获得的肌肉

可遗传变异的来源

  1. 基因突变：DNA分子中碱基对的替换、增添或缺失，导致基因结构的改变。是生物变异的根本来源。
  2. 基因重组：在有性生殖过程中，控制不同性状的基因重新组合。是生物变异的重要来源。
  3. 染色体变异：染色体数目或结构的改变。
💡 变异的意义
变异为生物进化提供了原材料。没有变异，就没有生物的多样性，自然选择也就无从谈起。可遗传的有利变异在世代积累中推动物种进化。

七、与小学教学的联系

🏫 教学建议
\"家庭遗传图谱\"：让学生调查自己家族的性状（如双眼皮/单眼皮、有耳垂/无耳垂、能卷舌/不能卷舌），画出简单的家系图，理解显隐性遗传规律。

\"DNA模型制作\"：用扭扭棒（pipe cleaners）或彩色毛线制作DNA双螺旋模型，直观理解碱基配对原则。

\"种子发芽对比实验\"：同一品种的种子分别种在不同环境中（光照/黑暗、充足/缺水），观察不可遗传变异的现象。
