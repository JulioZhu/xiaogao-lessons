电的基础

电流、电压、电阻是电学的三大基石。从静电到电路，从串联到并联，构建完整的电学知识框架。

学习目标

学完本课，你应该能够：

  1. 理解电荷的种类及相互作用规律
  2. 理解电流的形成条件及方向规定
  3. 理解电压的概念及常见电源电压值
  4. 理解电阻的概念及影响电阻大小的因素
  5. 掌握欧姆定律并能进行计算
  6. 区分串联电路和并联电路的特点
  7. 了解家庭电路的安全用电常识

一、电荷与静电

1. 两种电荷

⚡ 自然界只有两种电荷
  • 正电荷：用丝绸摩擦过的玻璃棒所带的电荷
  • 负电荷：用毛皮摩擦过的橡胶棒所带的电荷

2. 电荷间的相互作用

🔬 电荷间的作用规律
电荷种类 作用
正 — 正 相互排斥
负 — 负 相互排斥
正 — 负 相互吸引
💡 记忆口诀
同种电荷互相排斥，异种电荷互相吸引。

3. 静电现象

现象解释实例
摩擦起电电子从一个物体转移到另一个物体梳子梳头后吸纸屑、冬天脱毛衣有火花
感应起电带电体靠近导体，使导体两端出现等量异种电荷验电器的工作原理
静电屏蔽金属壳可以屏蔽外部电场高压作业工人穿金属丝织成的衣服

4. 原子结构与电荷

🔬 原子结构
原子核（带正电） — 位于中心，含质子和中子
核外电子（带负电） — 绕核运动

正常情况下：质子数 = 电子数 → 原子呈电中性
失去电子 → 带正电；得到电子 → 带负电
📌 元电荷
最小的电荷量叫元电荷：e = 1.6 × 10⁻¹⁹ C（库仑）
任何带电体的电荷量都是元电荷的整数倍。

二、电流

1. 电流的形成

💡 电流是电荷的定向移动形成的
  • 导体中有自由电荷（金属中是自由电子，溶液中是正负离子）
  • 导体两端有电压（电场力驱动电荷定向移动）

2. 电流的方向

⚠️ 考试高频考点
规定：正电荷定向移动的方向为电流方向。

因此，在金属导体中，电子定向移动的方向与电流方向相反。
这是考试常考的陷阱题！

3. 电流的单位

单位换算
1 A = 1000 mA = 1,000,000 μA
安培(A) → 毫安(mA) → 微安(μA)

4. 常见电流值

设备/现象电流（约）
手电筒中小灯泡约 0.2 A
家用电冰箱约 1 A
家用空调约 5 A
雷电可达 2 × 10⁴ A

5. 电流表的使用

📊 电流表使用规则
  1. 电流表必须串联在被测电路中
  2. 电流从正接线柱流入，从负接线柱流出
  3. 被测电流不能超过电流表的量程
  4. 绝对不允许不经过用电器直接将电流表接在电源两极上（会短路！）

三、电压

1. 电压的作用

🔋 电压是形成电流的原因
  • 电源是提供电压的装置
  • 有电压不一定有电流（还需要通路）
  • 有电流一定有电压

2. 电压的单位

单位换算
1 kV = 1000 V   |   1 V = 1000 mV

3. 常见电压值（考试必背！）

电源/场景电压
一节干电池1.5 V
一节蓄电池2 V
家庭电路220 V
安全电压（不高于）36 V
手机锂电池3.7 V
⚠️ 安全用电常识
不高于36V的电压才是安全电压。家庭电路220V远超安全电压，非常危险！

4. 电压表的使用

📊 电压表使用规则
  1. 电压表必须并联在被测电路两端
  2. 电流从正接线柱流入，从负接线柱流出
  3. 被测电压不能超过电压表的量程
  4. 电压表可以直接接在电源两极上，测量电源电压
📊 电流表 vs 电压表
比较项 电流表(A) 电压表(V)
连接方式 串联 并联
内阻 很小（相当于导线） 很大（相当于断路）
能否直接接电源 绝对不能 可以（测电源电压）

四、电阻

1. 电阻的概念

🔌 电阻表示导体对电流的阻碍作用
  • 符号：R
  • 单位：欧姆（Ω），还有千欧(kΩ)、兆欧(MΩ)
  • 导体电阻越小，导电性能越好；绝缘体电阻非常大
单位换算
1 MΩ = 1000 kΩ = 1,000,000 Ω

2. 影响电阻大小的因素

🔌 决定电阻大小的四个因素

R ∝ L / S   （材料、长度、横截面积、温度）

因素关系说明
材料不同材料电阻率不同铜的电阻率小于铁，所以铜导线导电更好
长度越长，电阻越大同一材料、粗细相同的导线，长的电阻大
横截面积越大，电阻越小粗导线比细导线电阻小
温度大多数导体温度越高电阻越大钨丝灯冷态电阻远小于热态电阻
💡 记忆要点
"长粗软"：导线越长电阻越大，越粗电阻越小，越"软"（电阻率小）的材料电阻越小。
注意：导体的电阻是导体本身的一种性质，与是否通电、电流大小、电压大小无关。

3. 变阻器

🔧 滑动变阻器
  • 原理：通过改变接入电路中电阻丝的长度来改变电阻
  • 符号：⏛（电路图中用带箭头的电阻符号表示）
  • 接法：一上一下串联接入电路
  • 作用：保护电路 + 调节电流/电压
📌 滑动变阻器的铭牌含义
例如"50Ω 2A"：
50Ω：最大电阻值为50欧姆
2A：允许通过的最大电流为2安培

五、欧姆定律

1. 欧姆定律

核心公式
I = U / R
电流 = 电压 ÷ 电阻
I — 电流（A）  |   U — 电压（V）  |   R — 电阻（Ω）
💡 理解欧姆定律
① 对同一导体（R不变）：电流与电压成正比（U越大，I越大）
② 对同一电压（U不变）：电流与电阻成反比（R越大，I越小）
③ 公式变形：U = IR（求电压）、R = U/I（求电阻，注意R不由U/I决定！）

2. 欧姆定律的应用

已知可求公式
电压 U、电阻 R电流 II = U / R
电流 I、电阻 R电压 UU = IR
电压 U、电流 I电阻 RR = U / I

3. 例题

📝 例题
一个电阻为 10Ω 的灯泡，接在 6V 的电源上，求通过灯泡的电流。

解：已知 U = 6V，R = 10Ω
由 I = U/R = 6V / 10Ω = 0.6 A

答：通过灯泡的电流为 0.6 安培。

六、串联电路与并联电路

1. 串联电路

🔗 串联电路特点
电源 → L₁ → L₂ → 回电源
  • 电流处处相等：I = I₁ = I₂
  • 总电压等于各部分电压之和：U = U₁ + U₂
  • 总电阻等于各电阻之和：R = R₁ + R₂
💡 串联电路的记忆
"串等流，压相加"：串联电路中电流处处相等，总电压等于各部分电压之和。
串联电阻越多，总电阻越大（R > R₁，R > R₂）。

2. 并联电路

⚡ 并联电路特点
  • 各支路两端电压相等：U = U₁ = U₂
  • 总电流等于各支路电流之和：I = I₁ + I₂
  • 总电阻的倒数等于各电阻倒数之和：1/R = 1/R₁ + 1/R₂
💡 并联电路的记忆
"并等压，流相加"：并联电路中各支路电压相等，总电流等于各支路电流之和。
并联电阻越多，总电阻越小（R < R₁，R < R₂）。
两个电阻并联的特殊公式：R = R₁ × R₂ / (R₁ + R₂)

3. 串联 vs 并联对比

比较项目串联电路并联电路
电流特点处处相等 I = I₁ = I₂干路电流等于各支路之和 I = I₁ + I₂
电压特点总电压等于各部分之和 U = U₁ + U₂各支路电压相等 U = U₁ = U₂
电阻特点总电阻增大 R = R₁ + R₂总电阻减小 1/R = 1/R₁ + 1/R₂
开关作用控制整个电路，位置不影响干路开关控制全部，支路开关只控本支路
用电器关系互相影响（一个断路，全部停止工作）互不影响（一个坏了，其他仍工作）
生活实例节日小彩灯家庭电路、教室灯光
🏠 家庭电路是典型的并联电路
家里各电器并联连接，所以可以独立开关、互不影响。
总开关在干路上，保险丝也在干路上。
插座也是并联在电路中的。

七、电功与电热

1. 电功

电功公式
W = UIt = Pt
电功 = 电压 × 电流 × 时间
W — 电功（J，焦耳）  |   P — 电功率（W）  |   t — 时间（s）

2. 电功率

电功率公式
P = W/t = UI
电功率 = 电功 ÷ 时间 = 电压 × 电流
📌 额定功率 vs 实际功率
额定电压(U额)：用电器正常工作时的电压
额定功率(P额)：在额定电压下工作时的功率
实际电压(U实)：实际加在用电器两端的电压
实际功率(P实)：在实际电压下的功率

当 U实 = U额 时，P实 = P额（正常工作）
当 U实 > U额 时，P实 > P额（可能烧坏）
当 U实 < U额 时，P实 < P额（发光暗淡）

3. 焦耳定律（电热）

焦耳定律
Q = I²Rt
电流通过导体产生的热量 = 电流平方 × 电阻 × 时间
⚠️ 电热与生活
利用电热：电热水器、电饭煲、电熨斗、电烙铁
防止电热危害：电脑CPU散热风扇、电视机散热孔、电动机外壳散热片

八、安全用电

🛡️ 安全用电原则
  1. 不接触低压带电体，不靠近高压带电体
  2. 更换灯泡、搬动电器前应断开电源
  3. 保险丝熔断后用同规格的保险丝替换，不能用铜丝/铁丝代替
  4. 发现有人触电，应立即切断电源或用绝缘体挑开电线

家庭电路的组成

🏠 家庭电路的连接顺序
进户线 → 电能表 → 总开关 → 保险装置 → 用电器/插座
元件作用安装位置
电能表测量消耗电能的多少进户线之后，总开关之前
总开关控制整个家庭电路的通断电能表之后，保险之前
保险丝电流过大时自动熔断，保护电路总开关之后
插座供 movable 用电器 plugged in与用电器并联
开关与灯泡控制灯泡通断开关接在火线上

家庭电路电流过大的原因

⚠️ 两大原因
  1. 短路：火线和零线直接连通，电阻极小，电流极大
  2. 过载（用电器总功率过大）：根据 I = P/U，总功率 P 越大，电流 I 越大
💡 试电笔的使用
手接触笔尾金属体，笔尖接触导线，氖管发光的是火线，不发光的是零线。
开关必须接在火线和用电器之间，这样断开开关后用电器不带电，更安全。
