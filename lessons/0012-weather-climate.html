天气与气候

天气和气候与我们息息相关——天气预报说的是"天气"，而"昆明四季如春"说的是"气候"。虽然只有一字之差，但它们背后的科学原理却大不相同。理解大气环流和气候类型，是掌握地球与宇宙科学领域的关键一环。

学习目标

学完本课，你应该能够：

  1. 准确区分"天气"和"气候"两个概念
  2. 说出气温的定义、测量方法和日较差/年较差的计算
  3. 理解等温线的含义及判读方法
  4. 掌握降水的三种主要类型及其形成条件
  5. 描述大气环流的三个圈层及其对应气压带和风带
  6. 说出世界主要气候类型的特征、分布及成因
  7. 理解影响气候的主要因素

一、天气 vs 气候

⚡ 核心区别
比较项天气气候
时间尺度短时间内（几分钟到几天）长时间（多年平均，通常30年统计）
变化性多变，不稳定相对稳定
描述内容阴晴、风雨、冷热等具体状况某地多年的平均大气状况
例子"今天上海小雨转多云，气温18-22℃""上海属亚热带季风气候，夏季高温多雨，冬季温和少雨"
🧠 记忆口诀
天气 = 今天的"心情"（短暂、多变）
气候 = 一个人的"性格"（长期、稳定）

"夜来风雨声，花落知多少" → 天气
"昆明四季如春" → 气候

二、气温

1. 气温的定义与测量

🌡️ 基本概念

气温（Air Temperature）是指空气的温度，单位通常为摄氏度（℃）。

  • 测量工具：温度计（通常放在百叶箱中，避免阳光直射和地面辐射影响）
  • 百叶箱高度：距地面 1.5米
  • 观测时间：通常每天观测 4次（8时、14时、20时、2时）

2. 气温的日变化与年变化

📈 日变化
指标说明
日最高气温通常出现在 午后14时左右（不是正午12时，因为地面需要时间加热大气）
日最低气温通常出现在 日出前后
日较差日最高温 − 日最低温。内陆地区日较差大，沿海地区日较差小
📊 气温日变化曲线示意
02时（最低）→ 06时（升温）→ 12时（继续升温）→ 14时（最高） → 18时（降温）→ 20时（继续降温）→ 次日02时（最低）

关键理解：气温变化的能量来源是 地面辐射，不是太阳直接加热空气。
太阳 → 加热地面 → 地面辐射 → 加热大气（"大地是天空的暖气片"）
📅 年变化
半球陆地最高月陆地最低月海洋最高月海洋最低月
北半球7月1月8月2月
南半球1月7月2月8月

年较差 = 最热月平均气温 − 最冷月平均气温。纬度越高、距海越远，年较差越大。

💡 为什么陆地比海洋升温快、降温也快？
因为水的 比热容 比陆地大得多。同样吸收1焦耳的热量，水温升高幅度远小于陆地。这就是沿海地区温差小、内陆温差大的原因。

3. 等温线

🗺️ 等温线图判读

等温线：在同一幅图上，把气温相同各点连接成的平滑曲线。

  • 等温线密集 → 温差大
  • 等温线稀疏 → 温差小
  • 等温线闭合 → 可能是高温中心或低温中心
  • 等温线与纬线平行 → 说明受纬度影响显著
🔑 等温线判读口诀
"凸高为低，凸低为高"

等温线向高值方向凸出 → 该处气温比同纬度低（可能是山地/寒流）
等温线向低值方向凸出 → 该处气温比同纬度高（可能是盆地/暖流）

三、降水

🌧️ 降水形成的两个基本条件
  1. 空气中含有足够的水汽和凝结核（尘埃等微粒）
  2. 空气温度下降到水汽能够凝结出来的程度（冷却过程）

1. 降水的三种主要类型

类型形成过程主要分布典型地区
对流雨
（Convective Rain）		 近地面空气强烈受热 → 强烈上升 → 冷却凝结 → 降水
特点：强度大、历时短、范围小，常伴有雷电		 赤道附近地区、夏季的中纬度大陆 亚马逊平原、我国南方夏季午后雷阵雨
地形雨
（Orographic Rain）		 湿润气流遇山脉阻挡 → 被迫抬升 → 冷却凝结 → 降水
迎风坡多雨，背风坡少雨（焚风效应）		 山地迎风坡 喜马拉雅山南坡、台湾火烧寮（中国雨极）
锋面雨
（Frontal Rain）		 冷暖气团相遇 → 暖气团被迫抬升 → 冷却凝结 → 降水
特点：范围广、历时长，是我国东部地区主要的降水形式		 中纬度地区 我国东部季风区的梅雨、华北雨季
⛰️ 地形雨示意图
迎风坡：湿润气流 → 上升 → 冷却 → 凝结 → 多雨
背风坡：气流下沉 → 增温 → 干燥 → 少雨（焚风效应）

关键公式：每上升100米，气温约下降 0.6℃（干绝热递减率）
例：山脚20℃，海拔2000米处 = 20 - 20×0.6 = 8℃
📊 世界降水量分布规律
  • 赤道附近：降水丰富（年均2000mm以上），因受热上升，多对流雨
  • 两极地区：降水稀少（年均200mm以下），因寒冷下沉，难以凝结
  • 中纬度沿海：降水较多，受海洋水汽影响
  • 中纬度内陆：降水较少，远离海洋
  • 大陆西岸：南北纬30°-40°之间地中海气候区冬季多雨；副热带地区少雨

四、大气环流与气压带、风带

🌀 大气环流：全球性的有规律的大气运动

大气环流是形成各地气候的基础。它包括 气压带、风带 和 季风 三部分。

1. 三圈环流与气压带

🌐 七个气压带（从赤道到极地）
序号气压带名称纬度位置气流运动干湿特征
1赤道低气压带0°附近受热上升湿润（多雨）
2副热带高气压带南北纬20°-30°堆积下沉干燥（少雨）
3副极地低气压带南北纬60°附近冷暖气团交汇上升湿润（多雨）
4极地高气压带南北纬80°-90°冷却下沉干燥（少雨）

2. 六个风带

💨 风带（由气压带之间的水平气压梯度力驱动）
风带名称从→到北半球风向南半球风向
信风带副热带高压 → 赤道低压东北信风东南信风
西风带副热带高压 → 副极地低压西南风西北风
极地东风带极地高压 → 副极地低压东北风东南风
💡 记忆技巧
"高气压带 → 干燥少雨；低气压带 → 湿润多雨"
"信风吹向赤道 → 干燥（但在迎风坡可带来降水）
西风带从海洋吹来 → 温和湿润（尤其大陆西岸）"

3. 气压带和风带的季节性移动

📅 移动规律
  • 随太阳直射点的移动而移动
  • 北半球夏季（7月）：整体向北移约5-10个纬度
  • 北半球冬季（1月）：整体向南移约5-10个纬度
  • 移动幅度：约 5-10个纬度
🔑 地中海气候的形成关键
地中海气候（南北纬30°-40°大陆西岸）：

夏季：受副热带高压控制 → 炎热干燥
冬季：受西风带控制 → 温和多雨

这是因为气压带风带随季节南北移动，导致同一地区在不同季节受不同气压带/风带控制。

五、世界主要气候类型

🌍 气候分类的核心依据

气候类型的判断遵循 "以温定带、以水定型" 的原则：

  • 以温定带：根据最冷月均温确定温度带
  • 以水定型：根据降水季节分配确定具体气候类型

1. 温度带判定标准

温度带最冷月均温代表气候类型
热带> 15℃热带雨林、热带草原、热带季风、热带沙漠气候
亚热带0℃ ~ 15℃亚热带季风、地中海气候
温带< 0℃（但最冷月 > -15℃）温带季风、温带海洋性、温带大陆性气候
寒带全年 < 0℃苔原、冰原气候

2. 世界主要气候类型速查表

气候类型分布规律气温特征降水特征典型植被
热带雨林气候 赤道附近（0°-10°） 全年高温（>25℃） 全年多雨（>2000mm） 热带雨林
热带草原气候 热带雨林南北两侧（10°-20°） 全年高温 干湿季分明 热带草原
热带季风气候 亚洲南部（10°-25°） 全年高温 旱雨季分明，雨季集中 热带季雨林
热带沙漠气候 南北回归线附近大陆西岸和内陆 全年高温 全年少雨（<250mm） 荒漠
亚热带季风气候 南北纬25°-35°大陆东岸 夏季高温，冬季温和（>0℃） 夏季多雨，冬季少雨 亚热带常绿阔叶林
地中海气候 南北纬30°-40°大陆西岸 夏季炎热，冬季温和 夏季干燥、冬季多雨（雨热不同期） 亚热带常绿硬叶林
温带季风气候 北纬35°-55°大陆东岸（东亚特有） 夏季高温，冬季寒冷（<0℃） 夏季多雨，冬季干燥 温带落叶阔叶林
温带海洋性气候 南北纬40°-60°大陆西岸 全年温和（冬无严寒、夏无酷暑） 全年湿润，降水均匀 温带落叶阔叶林
温带大陆性气候 温带大陆内部 冬冷夏热，年较差大 全年少雨 草原、荒漠
高原山地气候 高山高原地区 随海拔升高而降低 因地形而异 垂直变化明显
💡 考试高频对比
亚热带季风 vs 温带季风：最冷月均温是否低于0℃
亚热带季风 vs 地中海气候：雨热是否同期（季风=同期，地中海=不同期）
温带海洋性 vs 温带季风：降水是否均匀（海洋性=均匀，季风=夏季集中）
热带草原 vs 热带季风：降水量级不同，季风更多（年降水1500-2000mm vs 750-1000mm）

七、中国的气候特征

🇨🇳 中国气候的主要特点
  1. 季风气候显著：东部大部分地区属季风气候（热带/亚热带/温带季风）
  2. 雨热同期：夏季高温多雨，有利于农作物生长
  3. 气候复杂多样
  4. 大陆性气候分布广：西北内陆地区冬冷夏热，年较差大

八、易错点总结

❌ 常见错误
错误1认为最高气温出现在正午12时 → 错！出现在午后14时左右（地面加热大气需要时间）
错误2混淆地中海气候和亚热带季风气候 → 地中海：雨热不同期（夏干冬雨）；季风：雨热同期（夏雨）
错误3认为赤道地区都多雨 → 错！东非高原虽在赤道附近但因海拔高形成热带草原气候
错误4混淆信风和季风 → 信风是行星风带（常年），季风是季节性反转的风
错误5认为所有大陆西岸都是地中海气候 → 错！30°-40°才是，副热带地区（20°-30°）是沙漠气候
错误6认为寒流经过的地区一定寒冷 → 错！寒流主要影响降水（减湿），如撒哈拉沙漠延伸到西海岸（本格拉寒流）

九、与小学教学的联系

🏫 小学科学课程标准相关要点
  • 小学4-6年级要求学生能描述天气现象（阴晴、风雨、冷热等）
  • 能用简单表格统计气温数据，绘制气温变化图
  • 知道降水形式（雨、雪、冰雹等）
  • 了解本地的气候特点
  • 通过实验认识云的形成（模拟降雨）

作为高级教师候选人，你需要比小学教学要求更深一层——不仅要"描述"天气现象，还要能解释其背后的大气环流机制和气候成因。

十、随堂练习

请先独立思考，再点击选项查看答案和解析。

📝 课堂小测验（共8题）
1. 下列关于天气和气候的说法，正确的是：
✅ 正确！天气指短时间（几分钟到几天）的大气状况，多变；气候指多年（通常30年）的平均大气状况，相对稳定。
❌ 再想想。天气是短期的、多变的；气候是长期的、稳定的。B是正确答案。
2. 一天中，气温最高值通常出现在：
✅ 正确！虽然正午12时太阳辐射最强，但地面需要时间吸收热量再辐射给大气，所以气温最高值出现在午后14时左右。
❌ 记住：气温变化的能量来源是地面辐射，不是太阳直接加热空气。地面加热需要时间，所以最高温出现在14时左右，不是12时。
3. 下列哪种降水类型是由湿润气流遇山脉阻挡抬升形成的？
✅ 正确！地形雨是湿润气流遇到山脉等障碍物被迫抬升，冷却凝结形成的降水。迎风坡多雨，背风坡少雨。
❌ 地形雨的特征就是"遇山抬升"。对流雨是热空气上升形成的；锋面雨是冷暖气团相遇形成的。
4. 地中海气候的特征是：
✅ 正确！地中海气候最显著的特征是"雨热不同期"——夏季受副高控制干燥，冬季受西风带控制多雨。
❌ 地中海气候的关键特征是"夏干冬雨"，即雨热不同期。A是热带雨林，B是温带季风，C是温带海洋性。
5. 海拔每升高100米，气温大约下降：
✅ 正确！海拔每升高100米，气温约下降0.6℃。这是干绝热递减率，是计算山地气温的重要常数。
❌ 记住：海拔每升高100米，气温下降约0.6℃。例：山脚20℃，海拔2000米处 = 20 - 20×0.6 = 8℃。
6. 上海所属的气候类型是：
✅ 正确！上海位于秦岭-淮河以南（约北纬31°），属亚热带季风气候——夏季高温多雨，冬季温和少雨（最冷月均温>0℃）。
❌ 上海在秦岭-淮河以南，最冷月均温>0℃，是亚热带季风气候。温带季风气候最冷月<0℃（如北京）。
7. 下列哪个因素不是影响气候的主要因素？
