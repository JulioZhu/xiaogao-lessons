人体消化系统

从植物转向动物——学习人体如何获取能量。

学习目标

学完本课，你应该能够：

  1. 说出消化系统的组成（消化道 + 消化腺）
  2. 描述食物的消化过程及各器官的功能
  3. 列举主要消化酶及其作用底物和产物
  4. 理解小肠是消化吸收的主要场所
  5. 解释营养物质的吸收途径

一、消化系统的组成

消化系统由消化道和消化腺两部分组成。

1. 消化道（食物经过的通道）

🍽️ 食物之旅
口腔 → 咽 → 食道 → 胃 → 小肠 → 大肠 → 肛门

2. 消化腺（分泌消化液的腺体）

消化腺 位置 分泌的消化液 主要功能
唾液腺 口腔周围 唾液（含唾液淀粉酶） 初步消化淀粉
胃腺 胃壁 胃液（含盐酸、胃蛋白酶） 初步消化蛋白质
肝脏 腹腔右上部（人体最大腺体） 胆汁（不含消化酶！） 乳化脂肪
胰腺 胃下方 胰液（含多种消化酶） 消化三大营养物质
肠腺 小肠壁 肠液（含多种消化酶） 彻底消化营养物质
⚠️ 高频考点
胆汁不含消化酶，但能将脂肪乳化成微小的脂肪微粒，增大与消化酶的接触面积，属于物理性消化。这是考试常考的陷阱题！

二、三大营养物质的消化过程

1. 淀粉（糖类）的消化

淀粉 ──[唾液淀粉酶]──> 麦芽糖 ──[肠液/胰液中的酶]──> 葡萄糖
口腔开始消化 → 小肠彻底消化

2. 蛋白质的消化

蛋白质 ──[胃蛋白酶]──> 多肽 ──[肠液/胰液中的酶]──> 氨基酸
胃开始消化 → 小肠彻底消化

3. 脂肪的消化

脂肪 ──[胆汁乳化]──> 脂肪微粒 ──[肠液/胰液中的酶]──> 甘油 + 脂肪酸
仅在小肠消化（需要胆汁乳化）
营养物质 起始消化部位 彻底消化部位 最终产物 参与的消化液
淀粉口腔小肠葡萄糖唾液、胰液、肠液
蛋白质胃小肠氨基酸胃液、胰液、肠液
脂肪小肠小肠甘油 + 脂肪酸胆汁、胰液、肠液
🔑 关键理解
三大营养物质都是在小肠中被彻底消化的。小肠内有三种消化液（胆汁、胰液、肠液），是消化的"主战场"。

三、营养物质的吸收

消化后的营养物质通过消化道壁进入血液循环的过程叫吸收。

🎯 小肠为什么是吸收的主要场所？

四大适应特点：

  1. 长：小肠长约 5~6 米，吸收面积大
  2. 皱襞和绒毛：小肠内壁有环形皱襞，皱襞上有小肠绒毛，使吸收面积增大约 600 倍（可达 200 平方米）
  3. 薄：小肠绒毛壁和毛细血管壁都很薄，只由一层上皮细胞构成
  4. 密：小肠绒毛内有丰富的毛细血管和毛细淋巴管

吸收途径

营养物质 吸收部位 进入的血管
葡萄糖、氨基酸小肠毛细血管 → 门静脉 → 肝脏
甘油、脂肪酸小肠毛细淋巴管 → 淋巴循环 → 血液
水、无机盐、维生素小肠（主要）、大肠（部分水）毛细血管
📌 注意
大部分脂肪（甘油和脂肪酸）先进入毛细淋巴管，经淋巴循环最后才进入血液。这就是为什么高脂饮食后血液中会出现乳糜微粒。

四、胃的特殊功能

胃不仅是暂时储存食物的器官，还有重要的消化功能：

  • 胃酸（盐酸）：杀菌、激活胃蛋白酶原、使蛋白质变性便于消化
  • 胃蛋白酶：初步分解蛋白质为多肽（注意：不是完全分解为氨基酸）
  • 胃的蠕动：将食物与胃液混合形成食糜，推向小肠
⚠️ 考试高频陷阱
胃几乎不吸收营养物质！只能吸收少量的水、无机盐和酒精。这是很多同学容易搞混的点。

五、易错点总结

❌ 常见错误 ✅ 正确理解
胆汁含有消化酶胆汁不含消化酶，只能乳化脂肪（物理性消化）
淀粉在胃中被消化淀粉在口腔开始消化，胃中不消化淀粉
胃能吸收大量营养胃几乎不吸收，只有少量水和酒精
脂肪在大肠被消化脂肪仅在小肠被消化
蛋白质在胃中被彻底消化胃中只是初步消化为多肽，在小肠才被彻底消化为氨基酸
脂肪吸收直接进入血液脂肪先经淋巴循环，再进入血液

六、与小学教学的联系

作为小学科学教师，你需要向学生讲解消化系统的基本知识。以下是教学建议：

🏫 教学建议
食物旅行记：让学生扮演"一颗花生"，沿着消化道"旅行"，在每个器官描述发生了什么变化。这种角色扮演能帮助学生深刻理解消化过程。

小实验：用塑料袋装淀粉糊和唾液，放在37°C温水中，用碘液检验淀粉是否被分解。这个实验直观展示了唾液淀粉酶的作用。

健康饮食教育：结合消化系统，教育学生细嚼慢咽、规律饮食、不吃过期食物等良好习惯。

📝 课堂练习
1. 下列哪种消化液不含消化酶？
✅ 正确！胆汁由肝脏分泌，储存在胆囊中，不含任何消化酶，但能乳化脂肪。
❌ 不对。胆汁是唯一不含消化酶的消化液，它的作用是乳化脂肪。
2. 淀粉消化的最终产物是：
✅ 正确！淀粉 → 麦芽糖 → 葡萄糖。葡萄糖是最简单的糖，可以直接被吸收进入血液。
❌ 不对。淀粉的最终消化产物是葡萄糖。麦芽糖只是中间产物。
3. 蛋白质消化的起始部位是：
✅ 正确！蛋白质在胃中由胃蛋白酶开始消化，在小肠中被彻底分解为氨基酸。
❌ 不对。蛋白质消化的起始部位是胃（口腔消化淀粉，小肠消化脂肪）。
4. 脂肪消化的最终产物是：
✅ 正确！脂肪 → 脂肪微粒 → 甘油 + 脂肪酸。甘油和脂肪酸在小肠绒毛中被吸收。
❌ 不对。脂肪的最终产物是甘油和脂肪酸。
5. 下列哪项不是小肠适于吸收的结构特点？
✅ 正确！胆汁、胰液、肠液是消化液，与消化有关，与吸收无关。吸收的结构特点是长度、皱襞绒毛、壁薄和血管丰富。
❌ 不对。选项B说的是消化液，与消化功能有关，不是吸收的结构特点。
