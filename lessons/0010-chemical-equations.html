化学方程式入门

化学方程式是化学反应的"语言"。学会书写和配平，是理解物质变化的关键一步。

学习目标

学完本课，你应该能够：

  1. 理解化学方程式的含义和书写原则
  2. 掌握质量守恒定律并能用于配平
  3. 学会用最小整数法配平简单化学方程式
  4. 认识四种基本反应类型（化合、分解、置换、复分解）
  5. 理解化学方程式中各符号的意义
  6. 能写出初中阶段常见的化学方程式

一、质量守恒定律

1. 核心概念

⚖️ 质量守恒定律

参加化学反应的各物质的质量总和，等于反应后生成的各物质的质量总和。

这就是质量守恒定律。它是一切化学方程式配平的理论基础。

2. 为什么质量守恒？

🔬 微观解释
化学反应的本质是原子的重新组合。
反应前后：原子的种类不变、原子的数目不变、原子的质量不变
→ 所以物质的总质量不变
⚠️ 考试高频考点
质量守恒定律适用于一切化学反应，不适用于物理变化。
注意："守恒"指的是质量守恒，不是体积守恒、浓度守恒等其他量。

3. 容易混淆的概念

概念是否正确说明
化学反应前后，物质种类不变❌ 错物质种类一定改变（生成了新物质）
化学反应前后，元素种类不变✅ 对
化学反应前后，原子数目不变✅ 对
化学反应前后，分子数目不变❌ 错分子数目可能改变，如 2H₂ + O₂ → 2H₂O（3→2）
化学反应前后，分子种类不变❌ 错分子种类一定改变
化学反应前后，原子种类不变✅ 对

二、化学方程式的含义

1. 什么是化学方程式？

定义

用化学式来表示化学反应的式子

2. 以氢气燃烧为例

化学方程式
2H₂ + O₂ --(点燃)--> 2H₂O
📖 化学方程式的三层含义
  1. 质的方面：氢气和氧气在点燃条件下反应生成水
  2. 量的方面（粒子数）：每2个氢分子和1个氧分子反应生成2个水分子
  3. 量的方面（质量）：每4份质量的氢气和32份质量的氧气反应生成36份质量的水

3. 化学方程式中的符号

符号含义示例
+和、与（反应物之间或生成物之间）H₂ + O₂
-->生成（箭头指向生成物）--(点燃)--> H₂O
↑生成气体（反应物中没有气体时）Zn + H₂SO₄ --> ZnSO₄ + H₂↑
↓生成沉淀（溶液中反应）CuSO₄ + 2NaOH --> Cu(OH)₂↓ + Na₂SO₄
△加热2KClO₃ --(△,MnO₂)--> 2KCl + 3O₂↑
催化剂写在箭头上方或下方MnO₂ 作催化剂
💡 易错提醒
↑ 和 ↓ 的使用条件：
• ↑：生成物是气体，且反应物中没有气体时才标
• ↓：生成物是沉淀（不溶性固体），且反应在溶液中进行时才标

三、化学方程式的配平

1. 为什么要配平？

⚖️ 配平的依据：质量守恒定律

化学方程式两边，每种元素的原子个数必须相等。

配平就是在化学式前面配上适当的系数（最简整数比），使左右两边各元素的原子总数相等。

2. 最小整数配平法（最常用）

📝 配平步骤
  1. 找：找出左右两边原子个数最多的元素
  2. 定：确定该元素的最小公倍数，配出系数
  3. 调：根据已确定的系数，逐步调整其他元素的系数
  4. 查：检查左右两边各元素原子个数是否相等

3. 配平示例

📝 示例1：Fe + O₂ --(点燃)--> Fe₃O₄
第一步：找氧原子 → 左边2个，右边4个，最小公倍数4
第二步：O₂ 配系数2 → Fe + 2O₂ --> Fe₃O₄
第三步：Fe₃O₄ 中铁3个 → 3Fe + 2O₂ --> Fe₃O₄
第四步：检查 → 左：Fe=3, O=4；右：Fe=3, O=4 ✅

最终结果：3Fe + 2O₂ --(点燃)--> Fe₃O₄
📝 示例2：P + O₂ --(点燃)--> P₂O₅
第一步：找氧原子 → 左边2个，右边5个，最小公倍数10
第二步：O₂ 配5，P₂O₅ 配2 → P + 5O₂ --> 2P₂O₅
第三步：P₂O₅ 中磷4个 → 4P + 5O₂ --> 2P₂O₅
第四步：检查 → 左：P=4, O=10；右：P=4, O=10 ✅

最终结果：4P + 5O₂ --(点燃)--> 2P₂O₅
📝 示例3：CuSO₄ + NaOH --> Cu(OH)₂↓ + Na₂SO₄
观察法：右边Na₂SO₄中有2个Na → NaOH前配2
检查：Cu(OH)₂中有2个OH，2NaOH恰好提供2个OH ✅

最终结果：CuSO₄ + 2NaOH --> Cu(OH)₂↓ + Na₂SO₄
💡 配平口诀
左写反应物，右写生成物；
写好化学式，系数要配平；
中间连等号，条件不能丢；
生成气体标↑，沉淀要标↓。

四、四种基本反应类型

📊 四种基本反应类型速查
类型 通式 特征 实例
化合反应 A + B --> AB 多变一 C + O₂ --(点燃)--> CO₂
分解反应 AB --> A + B 一变多 2H₂O --(通电)--> 2H₂↑ + O₂↑
置换反应 A + BC --> AC + B 单质+化合物→新单质+新化合物 Zn + H₂SO₄ --> ZnSO₄ + H₂↑
复分解反应 AB + CD --> AD + CB 两种化合物交换成分 NaOH + HCl --> NaCl + H₂O

1. 化合反应

🔥 多变一
碳燃烧C + O₂ --(点燃)--> CO₂
硫燃烧S + O₂ --(点燃)--> SO₂
磷燃烧4P + 5O₂ --(点燃)--> 2P₂O₅
铁燃烧3Fe + 2O₂ --(点燃)--> Fe₃O₄
氢气燃烧2H₂ + O₂ --(点燃)--> 2H₂O
氧化钙与水CaO + H₂O --> Ca(OH)₂

2. 分解反应

💧 一变多
电解水2H₂O --(通电)--> 2H₂↑ + O₂↑
加热高锰酸钾2KMnO₄ --(△)--> K₂MnO₄ + MnO₂ + O₂↑
氯酸钾分解2KClO₃ --(△,MnO₂)--> 2KCl + 3O₂↑
碳酸分解H₂CO₃ --> H₂O + CO₂↑

3. 置换反应

⚔️ 单质换单质
锌与稀硫酸Zn + H₂SO₄ --> ZnSO₄ + H₂↑
铁与硫酸铜Fe + CuSO₄ --> FeSO₄ + Cu
氢气还原氧化铜H₂ + CuO --(△)--> Cu + H₂O
碳还原氧化铜C + 2CuO --(高温)--> 2Cu + CO₂↑

4. 复分解反应

🔄 成分互换
酸碱中和NaOH + HCl --> NaCl + H₂O
盐酸除锈Fe₂O₃ + 6HCl --> 2FeCl₃ + 3H₂O
碳酸钙与盐酸CaCO₃ + 2HCl --> CaCl₂ + H₂O + CO₂↑
生成沉淀BaCl₂ + Na₂SO₄ --> BaSO₄↓ + 2NaCl
💡 复分解反应发生的条件
生成物中必须有沉淀、气体或水（三者至少其一），反应才能发生。
这就是为什么酸碱中和能发生——生成了水！

五、常见气体的实验室制法

🧪 三种常见气体的制取
气体 反应原理 收集方法
O₂（氧气） 2KClO₃ --(△,MnO₂)--> 2KCl + 3O₂↑ 排水法或向上排空气法
CO₂（二氧化碳） CaCO₃ + 2HCl --> CaCl₂ + H₂O + CO₂↑ 向上排空气法（密度比空气大，溶于水）
H₂（氢气） Zn + H₂SO₄ --> ZnSO₄ + H₂↑ 排水法或向下排空气法（密度最小）

六、易错点总结

❌ 常见错误
错误1化学式写错（如把氧化铁写成FeO）→ 方程式一定错
错误2没有配平就写等号 → 违反质量守恒
错误3漏写反应条件（点燃、加热、催化剂等）
错误4不该标↑↓时标了（如反应物已有气体，生成气体不标↑）
错误5混淆"点燃"和"燃烧"——条件是"点燃"，现象是"燃烧"
错误6把质量守恒理解为体积守恒——气体反应前后体积可能变化

七、随堂练习

请先独立思考，再点击选项查看答案和解析。

📝 课堂小测验（共8题）
1. 下列变化中，遵循质量守恒定律的是：
✅ 正确！铁燃烧是化学反应，遵循质量守恒定律。冰融化、酒精挥发、玻璃破碎都是物理变化。
❌ 不对哦。质量守恒定律只适用于化学反应。铁燃烧是化学变化，其他三个都是物理变化。
2. 化学方程式 2H₂ + O₂ --(点燃)--> 2H₂O 中，氢气和氧气的分子个数比为：
✅ 正确！系数比就是分子个数比，H₂系数为2，O₂系数为1，比例为2:1。
❌ 再想想。化学方程式前面的系数表示分子个数比。2H₂ 和 O₂ 的系数分别是2和1。
3. 下列反应属于分解反应的是：
✅ 正确！一变多是分解反应的特征。2H₂O --> 2H₂ + O₂ 是一个物质变成两个物质。
❌ 分解反应的特征是"一变多"。A是化合反应，C是置换反应，D是复分解反应。
4. 实验室制取CO₂，最适合的方法是：
✅ 正确！CaCO₃(大理石) + 2HCl --> CaCl₂ + H₂O + CO₂↑。反应速率适中，便于收集。
❌ 大理石(CaCO₃)与稀盐酸反应是实验室制CO₂的标准方法。煅烧需要高温，木炭燃烧产物不纯。
5. 配平 Fe₂O₃ + CO --(高温)--> Fe + CO₂，正确的系数依次是：
✅ 正确！Fe₂O₃ + 3CO --(高温)--> 2Fe + 3CO₂。铁原子：左2=右2；氧原子：左3+3=6=右6；碳原子：左3=右3。
❌ 试试用观察法。Fe₂O₃中有3个O，需要3个CO来夺取这些O，生成3个CO₂，同时得到2个Fe。
6. 化学反应前后，一定不变的是：
✅ 正确！化学反应中分子种类和物质种类一定改变，分子数目可能改变，但原子种类和数目一定不变。
❌ 化学反应的本质是分子分裂成原子，原子重新组合成新分子。所以原子种类一定不变。
7. 下列符号中，表示"生成沉淀"的是：
✅ 正确！↓表示生成沉淀，↑表示生成气体，△表示加热，→表示生成。
❌ 记住：↑像气泡往上冒（气体），↓像沉淀往下沉（固体沉淀）。
8. 下列化学方程式书写完全正确的是：
✅ 正确！A未配平，B缺反应条件，C中H₂O是液体不应标↑，D完全正确。
❌ A没配平，B缺条件，C不该标↑（水是液态）。只有D同时满足配平、条件和符号规范。
